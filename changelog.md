# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-22

### Added
- **Global Schemas**:
    - Organization Schema (Trang chủ).
    - LocalBusiness Schema (Trang chủ).
    - WebSite Schema: Sitelinks Searchbox implementation.
- **Post Category Schemas**:
    - CollectionPage implementation.
    - ItemList implementation for post lists.
    - BreadcrumbList implementation.
- **Product Page Schemas**:
    - Product Schema: Includes Name, Image, Description, SKU, and Brand.
    - Offer Schema: Price, Currency (VND), Stock status.
    - AggregateRating Schema: Integrated with WooCommerce ratings (compatible with KK Star Ratings logic).
    - FAQPage Schema: Automated FAQ structure for products.
    - BreadcrumbList implementation for products.
- **Product Category Schemas**:
    - CollectionPage implementation.
    - ItemList implementation for product lists.
    - BreadcrumbList implementation.
